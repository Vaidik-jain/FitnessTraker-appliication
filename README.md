# 🏋️ Fitness Tracker Application (In Progress)
A web-based Fitness Tracker application designed to help users monitor their daily activities, track progress, and maintain a healthy lifestyle.

## 🚧 Project Status

⚠️ This project is currently under development.
New features and improvements are actively being added.

## 🎯 Objective

The goal of this project is to:

- Build a real-world application from scratch
- Improve backend and project structuring skills
- Integrate DevOps practices like containerization and CI/CD

## ✨ Features (Planned & In Progress)

- User authentication (Login/Signup)
- Activity tracking (steps, workouts, etc.)
- Progress monitoring dashboard
- REST API for handling user data
- Deployment-ready architecture
